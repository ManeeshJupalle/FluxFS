# FluxFS

Intelligent filesystem autopilot — watch, organize, deduplicate, and search your files automatically.

**Author:** Maneesh Jupalle — [maneeshreddy28@gmail.com](mailto:maneeshreddy28@gmail.com)

## Status

Phase 7 complete: activity log (`flux log`), status dashboard (`flux status`), JSONL rotation, and logging across scan/organize/dedup/watcher.

## Quick start

```bash
cargo build
cargo run -- init
cargo run -- config
```
